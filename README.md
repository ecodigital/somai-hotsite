# Hotsite SOMAI

## Dependências

 - node v0.10.x
 - npm v2.6.x

## Instalação

```
$ sudo npm install -g grunt-cli
$ git clone https://github.com/ecodigital/somai-hotsite.git
$ cd somai-hotsite
$ npm install
$ grunt build
```

## Rodando o app

```
$ cd dist
$ python -m SimpleHTTPServer
```

Acesse http://localhost:8000
